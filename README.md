## Description:
Matrix-calculator is an academic application designed to solve matrix operations and linear algebra problems in a clear and guided manner. It allows you to easily enter matrices, perform operations such as addition, subtraction, multiplication, transposition, determinant, and inverse, and tackle procedures such as row reduction Gauss, Gauss-Jordan and linear system solution/analysis.

## Run Locally

Clone the project

```bash
  git clone https://github.com/NaghellyM/matrix-calculator-fe.git
```

Go to the project directory

```bash
  cd matrix-calculator-fe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
